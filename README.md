# Vuetifier

Vuetify.js color and component examples.


## install
``` bash
yarn install
```

## run as develop mode
```bash
yarn dev
```

## generate for GitHub Pages
```bash
yarn generate
```

## deploy to GitHub Pages
```bash
yarn deploy
```

## generate for local hosting
```bash
yarn generate-local
```

## start local static server
```bash
yarn start-static
```

## lint
```bash
yarn lint
```
